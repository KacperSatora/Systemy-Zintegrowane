# Systemy Zintegrowane v0.3

Wymagania:

- Node.js v18+ or v20+

Aby uruchomić:

- ```npm install```
- ```npm run dev```
- w przeglądarce wyjść na podany w konsoli adres <http://localhost:5173>

Aby stworzyć .exe:

- ```npm run build```
- ```npm run dist```
- w folderze release\win-unpacked pojawi się plik Systemy Zintegrowane.exe
